First stable release.

Comprehensive conventions audit applied across the entire API surface: all
throwing/rejectable APIs return Result, functions with optional parameters split
into base + _with variants, boolean getters use is_ prefix, exhaustive type
safety in all FFI conversion helpers, and mutable operations documented.

New global_this module with alert, confirm, prompt, and close bindings.

Upgraded gossamer to 4.0.1. Developed and tested against Deno 2.7.
